# tarabanexperience-data

- Запускаем SQL скрипт внутри нашей БД (скрипт подходит, могу предположить, только для SQLite баз данных)
- Копируем результат, создаём файл рядом со скриптом с названием `toProccess.json`
- Запускаем скрипт, вуаля. Для отчётности можно переименовать затем ранее созданный файл под формат времени даты, когда этот тарабанск происходил.
